# Raj Nandan Sharma

## 简介

Raj Nandan Sharma 是一位软件工程师，自 2012 年开始编写代码，主要使用 Go、Svelte、Node.js 和 TypeScript。他的工作位于 fintech、AI 和开发者工具的交叉领域，同时构建 micro-SaaS 产品和自托管工具。

## 主要项目

- **Ellipsis** - TypeScript 库，实现 DOM 压缩用于 LLM（基于 arXiv 研究）
- **AI-powered Linear Agent** - 将 Linear tickets 转换为 PR 的 AI 代理
- **RAG Document Application** - 支持 PDF、DOCX、MD、HTML 等多种格式的文档问答系统

## 核心技术领域

1. **RAG Systems** - Sparse、Dense、Hybrid 检索策略
2. **AI Web Agents** - 浏览器自动化与 LLM 结合
3. **DOM Compression** - 为 LLM 优化网页内容

## 核心观点

### Agent Disciplines

- **Instruction Engineering** - 如何设计代理指令
- **Context Engineering** - 如何提供正确的上下文
- **Execution Engineering** - 如何执行和管理代理行为

### AGI 观点
