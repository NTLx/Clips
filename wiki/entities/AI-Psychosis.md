# AI Psychosis

## 定义

**AI Psychosis** 是 Andrej Karpathy 于 2026 年 4 月在 X 上提出的概念，描述一种存在于**付费使用前沿 agentic 模型进行技术工作的人群**中的认知状态：

> 当你把一个电脑终端交给这些模型时，你会看到它们轻松解决那些原本以为需要几天甚至几周才能完成的编程问题。正是这群人，会对 AI 的能力、能力增长的斜率，以及各种与网络安全相关的后果，赋予更大的分量和紧迫感。

## 关键数据点

| 维度 | 数据 |
|------|------|
| 提出者 | Andrej Karpathy (2026-04-09) |
| 定义群体 | 付费使用 Claude Code / Codex 的编程/数学/科研专业人员 |
| 核心现象 | AI 编程能力跃迁远超日常使用者体验 |
| 对比群体 | 使用免费 ChatGPT/语音助手的普通用户 |
| 编程占 ChatGPT 查询 | 约 4%（OpenAI + NBER 研究） |
| 非工作类查询 | 超过 73% |

## 前提与局限性

- **非临床术语**：不是心理学或精神病学意义上的"psychosis"，而是用来形容"极端震撼/脱离常见认知"的修辞表达
- **仅限技术领域**：仅指编程、数学、科研领域中 agentic 模型带来的体验，不代表 AI 在所有领域都有同等进步
- **时间窗口**：特指 2025-2026 年间能力跃迁最剧烈的阶段，可能随技术普及而消退
- **付费门槛**：依赖于每月 $200 级别的前沿模型访问权限，免费版/旧版本无法触发同等体验
