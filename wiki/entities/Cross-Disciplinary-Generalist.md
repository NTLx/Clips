# Cross-Disciplinary Generalist（跨学科通才）

> [!definition] 定义
> **Cross-Disciplinary Generalist** 是 Boris Cherny 在 Sequoia AI Ascent 2026 上提出的未来核心人才模型。传统"全栈工程师"仍然局限在工程领域内（iOS + Web + Server），而跨学科通才横跨工程、设计、产品、数据科学、甚至财务和用户研究。AI 抹平了编程能力差异后，深度的领域知识 + 广泛的跨职能能力成为最有价值的组合。

## 关键数据点

- **Claude Code 团队实践**：团队里每个人——工程经理、PM、设计师、数据科学家、财务、用户研究员——都在写代码。他们既是某个领域的专家，也是 coding builder
- **与传统"全栈"的区别**：旧的全栈 = 前端 + 后端 + DevOps；新的全栈 = 产品 + 工程 + 设计 + 数据
- **领域知识升值**："最会写会计软件的人是会计师，不是工程师——编程是简单部分，懂业务是难的部分"
- **角色边界模糊化**：PM 在写代码，工程师在做 PM 决策，设计师在改代码——专业职能从"职业身份"变为"能力工具箱"

## 前提与局限性

- 跨学科通才的前提是 AI 充分降低了每个领域的技术门槛——当前对某些领域（如复杂系统设计）仍然不够
- 广度 vs 深度的经典权衡仍然存在——最难的工程问题仍然需要专业深度
- 团队仍然需要某些形式的专门化——不是每个人都需要成为 everything

- [[Judgment]] — 跨职能决策需要更高维度的判断力
