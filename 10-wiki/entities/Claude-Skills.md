# Claude Skills

> [!definition] 定义
> Claude Skills 是给 AI 写的一份"岗位操作手册"，将工作方式——写作风格、代码规范、周报格式、数据分析维度等——全部写成一份文档，让 AI 读一遍就永远记住。下次只需说一句指令，AI 就自动按既定规矩来执行。

## 核心要点

### 与 Prompt 的本质区别

| 特性 | Prompt | Skill |
|------|--------|-------|
| 性质 | 一次性口头吩咐 | 可迭代的操作手册 |
| 记忆 | 每次需重新交代上下文 | 写入 SKILL.md，永久记住 |
| 价值 | 消耗品 | 资产，越用越值钱 |
| 迁移 | 无法跨平台 | 支持 Agent Skills 开放标准（agentskills.io） |

**核心洞察**: Prompt 是一次性消耗品，Skill 是越用越值钱的资产。

### 生态规模

- SkillsMP 平台聚合超过 **28 万个技能包**
- 每日以 **1 万+** 的速度新增
- 支持 **26+ 个平台和工具**：Claude Code、Cursor、Codex、GitHub Copilot 等

### 效率提升

Anthropic 官方研究结论：使用 Claude 辅助工作，**任务完成时间平均缩短约 80%**。

## 典型应用场景

### 内容创作者
- **写作风格生成器**: 分析 5-10 篇文章，自动生成专属风格卡片
- **内容策略规划**: 选题优先级评分（客户影响力 40%、内容匹配度 30%、搜索潜力 20%、资源投入 10%）
- **热点趋势猎手**: 扫描全网 30 天热点，数据驱动选题

### 产品经理
- **事前验尸 (Pre-Mortem)**: 设想项目失败，逆向推导失败原因
- **用户研究合成**: 访谈/问卷/工单转化为结构化洞察
- **冲刺规划**: 90 分钟准备工作缩到 15 分钟

### 程序员
- **全流程覆盖**: /plan → /tdd → /code-review → /commit 一条龙
- **发布流程优化**: 软件发版从 20 分钟缩到 2 分钟
